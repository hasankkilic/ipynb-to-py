Available tools
IPYNB → PY Converter

Upload a Jupyter Notebook (.ipynb) or paste its JSON content. We extract all code cells and generate a clean .py script.


Works entirely in your browser. No files are uploaded to any server.

JSON Beautifier

Paste your JSON and format it with indentation. Great for logs and API responses.

If JSON is invalid, you'll see an error message instead of formatted output.

CSV → JSON

Paste CSV data (first row as header) and convert it to JSON array.

Simple CSV parser (no advanced quoting). Good for quick conversions.
